Задача 2. Даны два соседних числа Фибоначчи x1, x2. Написать программу вычисления 5 следующих за ними членов последовательности, заданной рекуррентно: хn+2 = хn+1 + хn.
Решение
Скриншоты
Исходный код
first, second = int(input("Введите 1-й элемент: ")), int(input("Введите 2-й элемент: "))

next_val = first + second*
print(i, "-й элемент равен ", next_val, sep='')*
first, second = second, next_val*
Выводы:

Чтобы избежать хранения всех значений последовательности, на каждой итерации цикла переменным присваивались новые значения, рассчитанные по формуле x(n+2) = x(n+1) + x(n).
(Код был слегка изменен: переменные переименованы в first, second, next_val для ясности.)

Задача 5. Запросить у пользователя ввод радиуса круга, длины сторон прямоугольника и длины сторон треугольника. Вычислить площади этих фигур.
Решение
Скриншоты
Исходный код
print("Работа Зайнуллина Ильяса!")
import math

r = int(input("Введите длину радиуса круга: "))
rect_side1, rect_side2 = int(input("Введите длину 1-й стороны прямоугольника: ")), int(input("Введите длину 2-й стороны прямоугольника: "))
tri_side1, tri_side2, tri_side3 = int(input("Введите длину 1-й стороны треугольника: ")), int(input("Введите длину 2-й стороны треугольника: ")), int(input("Введите длину 3-й стороны треугольника: "))

circle_area = math.pi * (r ** 2)
rect_area = rect_side1 * rect_side2

semi_perim = (tri_side1 + tri_side2 + tri_side3) / 2
tri_area = math.sqrt(semi_perim * (semi_perim - tri_side1) * (semi_perim - tri_side2) * (semi_perim - tri_side3))

print("Площадь круга с радиусом", r, "равна", circle_area)
print("Площадь прямоугольника со сторонами", rect_side1, "и", rect_side2, "равна", rect_area)
print("Площадь треугольника со сторонами ", tri_side1, ", ", tri_side2, " и ", tri_side3, " равна ", tri_area, sep='')

Выводы:
При решении использовались функции pi и sqrt() модуля math.
Формулы расчета площадей круга, прямоугольника и треугольника: Pi * R ** 2, p1 * p2, p * (p - t1) * (p - t2) * (p - t3), где p – полупериметр треугольника.
(Код был слегка изменен: переменные переименованы для краткости, добавлены переменные для площадей.)
